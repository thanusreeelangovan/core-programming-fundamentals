# Student Result Analyzer

This Python project reads student marks from a CSV file, calculates total marks, average marks, pass/fail count, and identifies the topper.  

## Features
- Handles multiple students (example: 45 students)
- Calculates totals and averages for all subjects
- Dynamically identifies the topper
- Displays marks per subject
